# Yash Raj Kumar - AI Engineer Portfolio

A world-class AI Engineer portfolio built with Next.js, TypeScript, Tailwind CSS, Framer Motion, and Three.js.

## Features
- Dark mode with glassmorphism design
- 3D animated hero section with Three.js
- Responsive design (mobile, tablet, desktop)
- Smooth scroll animations with Framer Motion
- GitHub integration
- Contact form
- SEO optimized
- Accessible (WCAG compliant)

## Tech Stack
- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- Three.js / React Three Fiber
- Radix UI Primitives
- Lucide Icons

## Sections
- Hero with 3D animated background
- About with stats
- Skills with animated progress bars
- Experience timeline
- Education
- Projects with filtering
- GitHub contributions
- Research papers
- Achievements
- Certifications
- Contact form

## Getting Started
```bash
git clone https://github.com/YASHTHETIC/portfolio.git
cd portfolio
npm install
npm run dev
```
